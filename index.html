Welcome to the RTÉ Archives digital style guide.

This guide is to help with the dev, launch and run of the Archives site designed in 2014.

To get going, these files (index.html, styles, img assets) includes some basic styles you can modify, play around with, or totally destroy to get going.

The Grid

We use a responsive 12 column grid based on Foundation by Zurb. Max-width tops out at 1154px (not everyone has monster screens). Check out the sweet links below for more info on that.

This is a twelve column section in a row. Each of these includes a div.panel element so you can see where the columns are - it's not required at all for the grid.

Here are all the basic building blocks for the RTÉ Archives visual identity – typography, colour, icons and logos.

Type

Avenir Next – a bold typeface for the future and the past.Regular and italic 2007—2014

Avenir Next Condensed – for body copy, redesigned for onscreen. Reworked by Adrian Frutiger and Akira Kobayashi for Linotype, from 2004—2007.

Simple is a monospace typeface based reminiscent of screen and typewriter based fonts. Designed for use as a text font, it works very well at small point sizes.
